# Global Forest Analysis (Tableau)

Interactive Tableau dashboard analyzing global forest coverage, deforestation trends, and key drivers such as agriculture, population growth, and wood trade.

## Project Goals

This project explores:

• How global forest coverage has changed over the past decade  
• Which countries contribute most to global forest area  
• Relationships between forest coverage, agriculture, and population density  
• The role of wood trade in deforestation  

## Dashboard Features

- Global forest coverage map
- Country contribution analysis
- Forest change trends (2010–2021)
- Population vs forest relationship
- Wood trade analysis
